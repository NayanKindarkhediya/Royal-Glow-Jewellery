# Royal Glow Jewellery

A stylish and responsive jewellery website designed to showcase elegant jewellery collections with a premium and attractive user interface.

## ✨ Features
- Responsive jewellery website design
- Elegant homepage banner
- Jewellery collection showcase
- Clean and modern user interface
- Product sections for rings, necklaces, earrings, and more
- Mobile-friendly layout

## 🛠️ Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript

## 📂 Project Description
Royal Glow Jewellery is a frontend jewellery website project built to present jewellery products in a stylish and modern way.  
The project focuses on responsive design, clean layout, and attractive presentation of jewellery collections.
